# Customer Churn Prediction

A machine learning project that predicts whether a customer is likely to churn based on customer-related information. The project also includes a simple web interface for entering customer details and getting a churn prediction.

## Project Objective

The main objective of this project is to analyze customer data and build a machine learning model that can predict customer churn. This can help businesses identify customers who may leave their services.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Flask
- HTML
- Jupyter Notebook

## Project Structure

- `Customer churn prediction project.ipynb` - Contains data analysis, preprocessing and model development.
- `app.py` - Flask application used to run the prediction interface.
- `customer_churn_model.pkl` - Trained machine learning model.
- `templates/index.html` - Web interface for entering customer details and displaying predictions.
- `DATA/` - Contains the dataset used in the project.

## Features

- Customer churn prediction using machine learning.
- Data preprocessing and analysis.
- Trained model saved for future predictions.
- Simple web-based interface for entering customer information.
- Displays the churn prediction result.

## How to Run the Project

1. Download or clone the repository.
2. Install the required Python libraries.
3. Run `app.py`.
4. Open the local URL displayed in the terminal.
5. Enter the required customer details and click the prediction button.

## Output

The application predicts whether a customer is likely to churn based on the information entered by the user.
